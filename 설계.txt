💜 1. 시스템 개요

목적:
회사 내 프로젝트별 성과분석 및 리워드 계산 시스템.
→ 각 프로젝트의 진행 현황, 정산 내역, 팀원별 투입비용과 성과를 한눈에 파악.

핵심 특징:

UI 완성 중심 (데이터는 더미)

실제 데이터베이스 없이, JSON/mocks 기반으로 동작

추후 백엔드 연동 시 쉽게 이식 가능하도록 구조화

상단 필터바: 연도 / 프로젝트 / 팀 / 팀원 / 날짜 + 조회 버튼

아이콘 기능:

연필 ✏️ → 수정

자물쇠 🔒 → 저장

토글(보라색=활성, 회색=비활성)

🧱 4. 페이지별 상세 설계
① 대시보드

용도: 전체 현황 요약
구성(자리만):

총 매출 / 총 지출 / 총 영업이익 / 진행 프로젝트 수

프로젝트 상태 그래프

월별 매출 추이

② 스케줄 (Schedule)

목적: 팀원별 프로젝트 투입시간 기록

변경사항 반영:

입력 단위 HH:MM

예: 1:30, 2:00, 3:00

내부 저장 시 분 단위(90, 120, 180)

UI 구조:

필터: 연도 / 프로젝트 / 팀 / 팀원 / 기준일자 / 미입력 보기 토글

테이블:

열: 프로젝트명 | 팀 | 팀원 | (선택 날짜 기준 7일)

셀: TimeCell(HH:MM 입력 가능)

툴팁: 잘못된 입력 시 경고 표시

집계 로직 (더미계산):
참여일수 = 총투입분 ÷ 근로시간(분)
(기본 480분 = 8시간)

③ 프로젝트 (Projects)

목적: 프로젝트 마스터 관리

UI 구성:

필터: 연도 / 상태 / 구분 / 검색

컬럼:
프로젝트명 / 계약일 / 종료일 / 구분 / 상태 / 확정여부 / 계약금 / 잔여일수 / 실행률 / 수정

실행률 자동집계:

기본: 기간 기반
(오늘 - 시작일) / (종료일 - 시작일) × 100

투입 기반(추후 선택 가능):
(실제 투입일수 / 계획 투입일수) × 100

④ 정산 (Settlement)

목적: 프로젝트별 재무 요약

컬럼:
프로젝트명 / 계약일 / 종료일 / 구분 / 상태 / 확정 / 계약금 / 지출 / 영업이익 / 잔금 / 상세

로직(더미):

영업이익 = 계약금 - 지출

영업이익% = (영업이익 / 계약금) × 100

잔금 = 계약금 - 누적입금액

모달:

▸ [잔금·입금 내역 입력]

필터: 연도 / 프로젝트

입력: 입금일자 / 프로젝트명 / 입금액 / 비고

결과: 잔금 자동 계산 및 테이블 반영

문구: “* 모든 금액은 부가세 별도 기준입니다.”

⑤ 성과 (Performance)

목적: 팀/팀원별 성과 및 리워드 분석

필터: 연도 / 프로젝트 / 팀 / 팀원
열 구성:

프로젝트명 / 팀 / 팀원 / 계약금 / 계획 참여일수 / 참여율 / 1일 매출원가 / 1일 판관비 / 1일 투입비 / 참여일수(실적) / 총투입비용 / 성과

로직 (더미 계산 기준)

1일 매출원가 = 연봉 / (12 × 20.917)

1일 판관비 = (운영비 × (연봉 / 총연봉)) / 근무가능일수

1일 투입비 = 매출원가 + 판관비

총투입비용 = 1일 투입비 × 참여일수

성과 = 팀원 계약금 – 총투입비용

⑥ 팀원 (Members)

목적: 인사관리
구성:
팀원명 / 직급 / 팀 / 연봉 / 아이디 / 비밀번호 수정 / 권한 / 승인 / 비활성 / 저장 / 삭제

레벨:

총괄관리자 / 팀관리자 / 일반

특징:

승인/비활성 토글

자물쇠(저장), 휴지통(삭제)

⑦ 팀·직급 (Org)

좌측: 팀 목록
우측: 직급 목록
활성/비활성 토글 + 추가/삭제/저장

⑧ 운영비 (Opex)

목적: 월별 운영비 입력

컬럼:
연월 / 운영비 / 비고 / 저장 / 삭제

로직:

판관비 계산에 사용

연봉 비례 방식으로 직원별 분배

⑨ 휴일 및 근로시간 (Worktime)

좌측: 휴일 리스트

휴일 날짜 / 명칭 / 구분(자동추가)

우측: 근로시간

1일 근로시간(분) 입력 + 저장

기본값 480분

로직:

스케줄 계산 시 휴일 자동 제외

⑩ 기타설정 (Misc)

목적:

프로젝트 구분 관리

회사 기본정보

항목:

프로젝트 구분: (기획, 디자인, 설계, 마케팅, 어워드, BX 등)

회사명, 로고 업로드

⚙️ 5. 계산식 요약
항목	계산식
잔여일수	종료일 – 오늘
1일 매출원가	연봉 ÷ (12 × 20.917)
1일 판관비	(운영비 × 연봉비중) ÷ 근무가능일수
1일 투입비	매출원가 + 판관비
참여일수	총투입분 ÷ 근로시간(분)
총투입비용	1일 투입비 × 참여일수
성과	계약금 – 총투입비용
영업이익	계약금 – 지출총액
영업이익(%)	영업이익 ÷ 계약금 × 100
